# ICGFG-2026 Conference Website

An elegant conference website created for VIT Chennai’s International Conference on Graphs and Fuzzy Graphs (ICGFG-2026). The site presents the conference in a clean, formal, and easy-to-navigate format, bringing together the key details attendees need in one place.

It is optimized for a smooth browsing experience, quick access to conference information, and clear presentation across different screen sizes. The layout is built to support a broad audience of students, faculty, researchers, and event visitors without feeling crowded or difficult to follow.

## Live Site

https://chennai.vit.ac.in/files/icgfg2026/

## What It Presents

- Conference overview and institutional identity
- Registration information and participation details
- Important dates and announcements
- Committee and speaker-related sections
- Venue and contact information

## Experience

- Optimized for readability and quick navigation
- Designed to serve a wide conference audience
- Presented in a professional format suitable for an official academic event
